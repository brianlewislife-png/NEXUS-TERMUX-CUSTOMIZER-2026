## Overview

**NEXUS TERMUX CUSTOMIZER 2026** is a modern, complete terminal customization tool built exclusively for **Termux on Android**. It transforms your mobile terminal into a personalized, organized, and futuristic environment.

Run the installer, answer two simple questions, and your entire Termux environment is automatically personalized.

---

<table>
<tr>
<td>

- Complete Termux customization
- Automatic ASCII banner creation
- Personalized username & nickname

- Shell environment configuration

- Essential package installer
- Configuration backup & restore
- Built-in update system

## Quick Install

Open **Termux** and run:

```bash
# Install git if you don't have it
pkg install git

# Clone the repository
git clone https://github.com/brianlewislife-png/NEXUS-TERMUX-CUSTOMIZER-2026.git

# Enter the directory
cd NEXUS-TERMUX-CUSTOMIZER-2026

# Run the installer
bash install.sh
```

After installation, simply run:

```bash
nexus
```

## What It Does

On first run, NEXUS asks for your name and nickname. Then you can:

1. **Manual mode** — Choose your name, nickname, and PS1 style
2. **Banner creator** — Generate a personalized ASCII banner (4 styles + custom)
3. **Theme selector** — Pick from 6 color themes (Cyber, Neon, Ocean, Matrix, Sunset, Dracula)
4. **Advanced settings** — Manage aliases, install packages, configure PS1, MOTD, backup/restore, update
5. **Auto mode** — One-click setup with Cyber theme, essential aliases, classic PS1, and welcome MOTD
